## Requirements

- [Node.js](https://nodejs.org/en/) = v18.12.0
- [Yarn](https://yarnpkg.com/en/) = 1.22.19
- [Npm](https://www.npmjs.com/) = 8.19.2

## Installation

```bash
$ npm install
```

## Running the app

```bash
# development
$ yarn run start

# watch mode
$ yarn dev

# production mode
$ yarn run start:prod
```

## Generate

```bash
# generate module
$ nest generate module modules/<module-name>

# generate controller
$ nest generate controller modules/<module-name>

# generate service
$ nest generate service modules/<module-name>

# generate entity
$ nest generate entity modules/<module-name>

or

# Custom generate
$ npm run plop
```
